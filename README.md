## 1. Overview
Aporia Discovery is an AI-driven framework for **Inverse Molecular Design**. 
It goes beyond simple generation: It validates chemical stability, solubility, and **dynamic functionality** in real-time.

## 2. Highlight: The Perfect PROTAC Linker
We tasked the engine to design a linker for a Proteolysis Targeting Chimera (PROTAC).
*   **Result:** CCNOCCOCCNC#CCCN
*   **Physics:** 16 Atoms, Solubility Ratio 2.4
*   **Dynamics:** Monte Carlo simulation confirms **40.57%** probability of target engagement ("Kissing Zone").

> **Significance:** A probability >30% is considered excellent for clinical candidates.
